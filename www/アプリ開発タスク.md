# アプリ開発タスク

## 開発ルール
- **GitHub push はこまめに実行する**（指示完了・ファイル更新の都度即push）
- ユーザーから来た指示を日付+番号で管理し、完了後も履歴として残す
- 新しい指示が来たら即番号付きで追加
- ステータス：🔄 未着手 / 🔧 作業中 / ✅ 完了 / ⏸️ 保留
- コミットIDも併記して追跡可能に
